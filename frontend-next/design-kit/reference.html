Four-tier signal system. --accent positive, --gold warn/urgent, --threat destructive actions and terminal failure (delete, revoke, bounced), --info system work in progress (syncing, indexing). All four tiers are available on chips, banners, and buttons — but destructive and info are rare by rule. Max one threat chip per row, always pair info chips with motion. Destructive color is a low-chroma oxblood; informational is a desaturated slate-blue. No spectral red.

11 · State chips

Four-tier signal, sparingly used.

Latin · default mono
NEW QUOTING REPLY NOW BOUNCED SYNCING DRAFT SENT
CJK · body font, no uppercase
新建 报价中 下滑中 增长中 立即回复 退回 同步中 多次查看 已发送

增长中 (competitor gaining = threat) goes gold. 下滑中 (competitor losing share = opportunity for the user) goes accent. The remapping inverts the "red = bad" reflex but matches the user's actual interest.

Rules for the new tiers. chip-threat is for terminal failure row states only — bounced email, revoked access, failed payment. One per row, max. Never for "needs attention" (use gold) or stale data (use gold). chip-info is for in-progress system work — syncing, indexing, processing. Always pair with a dot or spinner to signal motion. Never for static informational states.
